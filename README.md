**Stableyield** 是专为 Polymarket 预测市场打造的量化交易与数据服务客户端。我们通过严谨的数学对冲模型与毫秒级执行引擎，为用户提供稳健的收益基础设施。

---

## 🚀 核心策略与功能

目前客户端已集成以下核心自动化交易模块：

### 1. V1.0 结算套利 (Settlement Arbitrage)
* **逻辑：** 专注于已结束事件的结算机制，捕捉市场关闭后的价格回归机会。
* **优势：** 风险极低，执行高度自动化。

### 2. V2.0 二元互补套利 (Binary Complementary Arbitrage)
* **逻辑：** 实时监控盘口，当出现 `YES + NO < 1` 的定价偏差时，瞬时执行买入。
* **优势：** 纯粹的数学性获利机会，无视市场涨跌方向。

### 3. 数据终端与信号
* 提供对标行业领先平台的高阶数据看板。
* 支持自定义策略预警与信号推送。
